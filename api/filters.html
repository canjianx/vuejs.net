过滤器

capitalize

‘abc’ => ‘Abc’

uppercase

‘abc’ => ‘ABC’

lowercase

‘ABC’ => ‘abc’

currency

  • 可以带一个可选参数

12345 => $12,345.00

可以传入一个符号 (默认是 $).

pluralize

  • 至少有一个参数

基于过滤的值来使参数复数化。如果只有一个参数，复数化的形式只是简单的加一个”s”。 如果有一个以上的参数，那么参数将会当成相对应的一个，两个，三个等等复数化的形式相对应的数组。如果复数化的内容超过了参数长度，它将使用数组中的最后一个值。

例子:

1 => ‘1 item’
2 => ‘2 items’

结果是:

1 => ‘1st’
2 => ‘2nd’
3 => ‘3rd’
4 => ‘4th’
5 => ‘5th’

key

  • 这个过滤器只工作在v-on里
  • 只有并只能有一个参数

  • tab

  • esc
  • up
  • down
  • left
  • right
